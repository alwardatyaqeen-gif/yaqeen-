# yaqeen-
تم تنفيذ هذا الواجب ضمن مادة الوحدة 21 مقدمة إلى الذكاء الاصطناعي بهدف تخطيط وتطوير حل عملي باستخدام تقنيات التعلم الموجه وتطبيقه على مشكلة التنبؤ بالمبيعات. المشروع يوضح مراحل بناء نموذج ذكاء اصطناعي بدءا من جمع البيانات وتحليلها وتنظيفها ثم معالجة الخصائص وتدريب النموذج وتقييم أدائه وتحسينه بشكل تكراري مع التركيز على منع تسرب البيانات وضمان موثوقية النتائج. تم تنفيذ الحل باستخدام لغة البرمجة Python داخل بيئة Google Colab مع استخدام مكتبات تحليل البيانات والتعلم الآلي وتوثيق جميع المراحل بالأكواد والمخرجات والرسوم البيانية لدعم عملية التقييم والتحقق من تحقيق معايير النجاح المطلوبة.
